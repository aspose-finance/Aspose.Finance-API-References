## ProfileRequest class

Una richiesta di profilo indica quali componenti di profilo desidera un client. Indica anche qual è la capacità di routing del client. I profili restituiti dall'FI devono essere compatibili con lo stile di instradamento richiesto, altrimenti il server restituisce un errore.
