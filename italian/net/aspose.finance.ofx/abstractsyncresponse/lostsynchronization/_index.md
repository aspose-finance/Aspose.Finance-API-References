## AbstractSyncResponse.LostSynchronization property

Sì se il token nella richiesta di sincronizzazione è più vecchio della prima voce nella tabella della cronologia del server. In questo caso, alcune risposte sono andate perse. No se il token nella richiesta di sincronizzazione è più recente o corrisponde a un token nel server tabella della cronologia.
