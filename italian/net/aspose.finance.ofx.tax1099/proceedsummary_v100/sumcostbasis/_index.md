## ProceedSummary_V100.SumCostBasis property

Ottiene o imposta la base dei costi. Questa è la base del costo totale per tutte le transazioni riportate in questo riepilogo. Per le transazioni del codice A e D del modulo 8949, this dovrebbe essere la base dei costi segnalata all'IRS. Per tutte le altre transazioni, la base dei costi dovrebbe essere la base dei costi nota dal fornitore.

public string SumCostBasis { get; set; }
