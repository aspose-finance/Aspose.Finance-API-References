## Aspose.Finance для .NET

{{% alert color="primary" %}}

![Логотип продукта Aspose.Finance for .NET](../home_1.png)


Aspose.Finance как чистая библиотека .NET обеспечивает гораздо лучшую производительность и простоту использования для управления форматами, связанными с финансами, такими как XBRL, iXBRL, OFX. Финансовый API является расширяемым, простым в использовании и компактным, а также предоставляет все стандартные функции, поэтому разработчики пишут меньше кода для выполнения стандартных операций.
