## ProceedSummary_V100.SumCostBasis property

Получает или задает базовую стоимость. Это общая стоимостная основа для всех транзакций, указанных в этой сводке. Для транзакций формы 8949, код A и D, эта должна быть стоимостной базой, сообщаемой в IRS. Для всех других транзакций стоимостная база должна быть базовая стоимость, известная поставщику.

public string SumCostBasis { get; set; }
