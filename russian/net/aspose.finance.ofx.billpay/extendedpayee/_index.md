## ExtendedPayee class

Класс сообщает идентификатор получателя платежа клиенту. Он также содержит параметры дня обработки для получателя платежа. Он может быть отправлен клиенту для любого получателя, чьи параметры дня обработки отличаются от значений по умолчанию процессора, даже для получателей без идентификатора получателя.
