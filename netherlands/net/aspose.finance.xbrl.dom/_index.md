## Klassen

| Klas | Beschrijving |
| --- | --- |
| [Attr](./attr/) | De Attr-interface vertegenwoordigt een attribuut in een Element-object. Meestal worden de toegestane waarden voor het kenmerk gedefinieerd in een schema dat aan het document is gekoppeld. |
| [CharacterData](./characterdata/) | De CharacterData breidt Node uit met een set attributen en methoden voor toegang tot karaktergegevens in de DOM. |
| [Comment](./comment/) | Neemt over van CharacterData en vertegenwoordigt de inhoud van een opmerking. |
| [Document](./document/) | Het document vertegenwoordigt het volledige inline xbrl-document. Conceptueel is het de root van de documentboom en biedt het de primaire toegang tot de documentgegevens. |
| [Element](./element/) | De Element-interface vertegenwoordigt een element in een inline xbrl-document. |
| [ElementList](./elementlist/) | De lijst met elementen. |
| [NamedNodeMap](./namednodemap/) | Vertegenwoordigt verzamelingen attributen die op naam toegankelijk zijn. |
| [Node](./node/) | De klasse Node is het primaire gegevenstype voor het gehele documentobjectmodel. Het vertegenwoordigt een enkel knooppunt in de documentboom. |
| [NodeList](./nodelist/) | De lijst met knooppunten. |
| [ProcessingInstruction](./processinginstruction/) | De klasse vertegenwoordigt een "verwerkingsinstructie", die in XML wordt gebruikt als een manier om processorspecifieke informatie in de tekst van het document te bewaren. |
| [Text](./text/) | De klasse vertegenwoordigt de tekstuele inhoud. |
