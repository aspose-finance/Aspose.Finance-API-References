## StatementTransaction class

Deze klasse beschrijft een enkele transactie. Het identificeert het type transactie en de datum waarop het is geboekt. De klasse kan ook aanvullende informatie bieden om de klant te helpen de transactie te herkennen: chequenummer, naam van de begunstigde en memo. De transactie kan een standaard industriële code hebben die een klant kan gebruiken om de transactie te categoriseren.
