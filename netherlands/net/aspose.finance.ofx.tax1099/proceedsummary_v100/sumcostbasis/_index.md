## ProceedSummary_V100.SumCostBasis property

Haalt of stelt de kostenbasis in. Dit is de totale kostenbasis voor alle transacties die in dit overzicht worden gerapporteerd. Voor formulier 8949 Code A- en D-transacties moet dit de kostenbasis zijn zoals gerapporteerd aan de IRS. Voor alle andere transacties moet de kostenbasis de bekende kostenbasis zijn door de aanbieder.

public string SumCostBasis { get; set; }
