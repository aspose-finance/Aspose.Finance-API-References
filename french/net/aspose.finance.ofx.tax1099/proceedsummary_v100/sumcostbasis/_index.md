## ProceedSummary_V100.SumCostBasis property

Obtient ou définit la base de coût. Il s'agit de la base de coût total pour toutes les transactions déclarées dans ce résumé. Pour les transactions du formulaire 8949 Code A et D, this doit être la base de coût telle que rapportée à l'IRS. Pour toutes les autres transactions, la base de coût doit être la base de coût connue par le fournisseur.
