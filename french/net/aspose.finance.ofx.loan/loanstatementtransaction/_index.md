## LoanStatementTransaction class

Cette classe décrit une seule transaction. Il identifie le type de transaction et la date à laquelle elle a été enregistrée. La classe peut également fournir des informations supplémentaires pour aider le client à reconnaître la transaction : numéro de chèque, nom, mémo et images.
