## クラス

| クラス | 説明 |
| --- | --- |
| [Attr](./attr/) | Attr インターフェイスは、Element オブジェクトの属性を表します。通常、属性の許容値は、ドキュメントに関連付けられたスキーマで定義されます. |
| [CharacterData](./characterdata/) | CharacterData は、DOM 内の文字データにアクセスするための一連の属性とメソッドで Node を拡張します。 |
| [Comment](./comment/) | CharacterData から継承し、コメントの内容を表します。 |
| [Document](./document/) | Document は、インライン xbrl ドキュメント全体を表します。概念的には、ドキュメント ツリーのルートであり、ドキュメントのデータへの主要なアクセスを提供します。 |
| [Element](./element/) | Element インターフェイスは、インライン xbrl ドキュメント内の要素を表します。 |
| [ElementList](./elementlist/) | 要素のリスト. |
| [NamedNodeMap](./namednodemap/) | 名前でアクセスできる属性のコレクションを表します。 |
| [Node](./node/) | Node クラスは、Document オブジェクト モデル全体の主要なデータ型です。これは、ドキュメント ツリー内の単一のノードを表します。 |
| [NodeList](./nodelist/) | ノードのリスト. |
| [ProcessingInstruction](./processinginstruction/) | クラスは「処理命令」を表し、ドキュメントのテキストにプロセッサ固有の情報を保持する方法として XML で使用されます。 |
| [Text](./text/) | クラスはテキスト コンテンツを表します。 |
