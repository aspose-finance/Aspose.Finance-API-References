## MatchInfo.StartOfYear property

雇用主の貢献度の最大値がいつリセットされるかを取得または設定します。一部の 計画には、暦年ではなく ではなく 会社の会計年度に基づいた最大値があります。省略した場合は暦年を想定します。 月と日(MMDD)のみが使用されます。年(YYYY)と時刻は無視されます.

```csharp
public string StartOfYear { get; set; }
```
