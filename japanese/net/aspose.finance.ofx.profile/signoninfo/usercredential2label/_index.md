## SignonInfo.UserCredential2Label property

ユーザー資格情報のテキスト プロンプトを取得または設定します。存在する場合、USERID、USERPASS、および USERCRED1 に加えて、4 番目の資格情報 (USERCRED2) が必要です。存在する場合、UserCredential1Label も存在する必要があります。

public string UserCredential2Label { get; set; }
