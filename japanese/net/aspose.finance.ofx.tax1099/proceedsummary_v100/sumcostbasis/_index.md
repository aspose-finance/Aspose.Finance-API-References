## ProceedSummary_V100.SumCostBasis property

コスト基準を取得または設定します。これは、この概要で報告されたすべての取引の総原価基準です。フォーム 8949 コード A および D の取引の場合、this が IRS に報告された原価基準になります。他のすべての取引の場合、原価基準は既知の原価基準になります。プロバイダーによる.

```csharp
public string SumCostBasis { get; set; }
```
