## ExtendedPayee class

クラスは、受取人 ID をクライアントに伝えます。これには、受取人の処理日パラメーターも含まれます。受取人 ID のない受取人であっても、処理日パラメーターがプロセッサーのデフォルト値と異なる受取人のクライアントに送信できます。
