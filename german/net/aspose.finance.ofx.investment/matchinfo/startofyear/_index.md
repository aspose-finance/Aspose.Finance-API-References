## MatchInfo.StartOfYear property

Ermittelt oder setzt die Angabe, wann der maximale Arbeitgeberbeitrag zurückgesetzt wird. Einige -Pläne haben ein Maximum basierend auf dem Geschäftsjahr des Unternehmens statt statt dem Kalenderjahr. Nehmen Sie das Kalenderjahr an, wenn es weggelassen wird. Es werden nur der -Monat und -Tag (MMTT) verwendet; Jahr (JJJJ) und Zeit werden ignoriert.

public string StartOfYear { get; set; }
