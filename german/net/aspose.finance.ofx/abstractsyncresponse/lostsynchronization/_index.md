## AbstractSyncResponse.LostSynchronization property

Ja, wenn das Token in der Synchronisierungsanforderung älter ist als der früheste Eintrag in der Verlaufstabelle des Servers. In diesem Fall sind einige Antworten verloren gegangen. Nein, wenn das Token in der Synchronisierungsanforderung neuer ist als oder mit einem Token auf dem Server übereinstimmt Verlaufstabelle.
