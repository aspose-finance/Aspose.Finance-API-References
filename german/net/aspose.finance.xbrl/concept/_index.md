## Concept class

Konzepte werden auf zwei äquivalente Weisen definiert. Im syntaktischen Sinne ist ein Konzept eine XML-Schema-Elementdefinition, definiert das Element, das in der Element-Element-Ersetzungsgruppe oder in der Tupel-Element-Ersetzungsgruppe sein soll. Auf semantischer Ebene ist ein Konzept eine Definition einer Art von Tatsache, die über die Aktivitäten oder die Art einer Geschäftstätigkeit berichtet werden kann.
