## RecurringIntraCancellationRequest.CancelPending property

Ruft das Flag für anstehendes Abbrechen ab oder setzt es. Wenn ja, sollte der Server alle ausstehenden und nicht gespawnten Übertragungen abbrechen. Wenn nein, sollte der Server nur das Modell (und unspawned Transfers) abbrechen.
