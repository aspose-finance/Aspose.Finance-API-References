## ProceedSummary_V100.SumCostBasis property

Ruft die Kostenbasis ab oder legt sie fest. Dies ist die Gesamtkostenbasis für alle in dieser Zusammenfassung gemeldeten Transaktionen. Für Transaktionen gemäß Formular 8949 Code A und D sollte this die Kostenbasis sein, wie sie dem IRS gemeldet wird. Für alle anderen Transaktionen sollte die Kostenbasis die bekannte Kostenbasis sein vom Anbieter.
