## RecurringPaymentCancelRequest.CancelPending property

Ruft das Flag "Abbrechen anstehend" ab oder setzt es. Wenn wahr, sollte der Server alle ausstehenden und unspawned Zahlungen stornieren. Wenn falsch, sollte der Server nur das Modell (und nicht gespawnte Zahlungen) stornieren.
