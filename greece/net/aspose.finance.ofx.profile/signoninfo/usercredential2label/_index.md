## SignonInfo.UserCredential2Label property

Λαμβάνει ή ορίζει το μήνυμα κειμένου για τα διαπιστευτήρια χρήστη. Εάν υπάρχει, απαιτείται ένα τέταρτο διαπιστευτήριο (USERCRED2) επιπλέον των USERID, USERPASS και USERCRED1. Εάν υπάρχει, πρέπει επίσης να υπάρχει UserCredential1Label.

```csharp
public string UserCredential2Label { get; set; }
```
