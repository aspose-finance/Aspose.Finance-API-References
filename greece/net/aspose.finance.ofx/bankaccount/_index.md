## BankAccount class

Το OFX χρησιμοποιεί τα συγκεντρωτικά στοιχεία τραπεζικών λογαριασμών για την αναγνώριση ενός λογαριασμού σε ένα FI. Τα συγκεντρωτικά στοιχεία περιέχουν αρκετές πληροφορίες για τον μοναδικό προσδιορισμό ενός λογαριασμού για τους σκοπούς της λήψης αντίγραφου κίνησης, της πληρωμής λογαριασμών και της μεταφοράς χρημάτων.

```csharp
public class BankAccount : AbstractAccount
```
