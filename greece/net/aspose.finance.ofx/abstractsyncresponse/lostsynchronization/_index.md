## AbstractSyncResponse.LostSynchronization property

Ναι, εάν το διακριτικό στο αίτημα συγχρονισμού είναι παλαιότερο από την παλαιότερη καταχώριση στον πίνακα ιστορικού του διακομιστή. Σε αυτήν την περίπτωση, ορισμένες απαντήσεις έχουν χαθεί. Όχι εάν το διακριτικό στο αίτημα συγχρονισμού είναι νεότερο ή ταιριάζει με ένα διακριτικό στον διακομιστή πίνακας ιστορικού.

```csharp
public bool LostSynchronization { get; set; }
```
