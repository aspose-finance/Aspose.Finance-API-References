## Τάξεις

| Τάξη | Περιγραφή |
| --- | --- |
| [Attr](./attr/) | Η διεπαφή Attr αντιπροσωπεύει ένα χαρακτηριστικό σε ένα αντικείμενο Element. Συνήθως οι επιτρεπόμενες τιμές για το χαρακτηριστικό ορίζονται σε ένα σχήμα που σχετίζεται με το έγγραφο. |
| [CharacterData](./characterdata/) | Το CharacterData επεκτείνει τον κόμβο με ένα σύνολο χαρακτηριστικών και μεθόδων για την πρόσβαση σε δεδομένα χαρακτήρων στο DOM. |
| [Comment](./comment/) | Κληρονομεί από το CharacterData και αντιπροσωπεύει το περιεχόμενο ενός σχολίου. |
| [Document](./document/) | Το Έγγραφο αντιπροσωπεύει ολόκληρο το ενσωματωμένο έγγραφο xbrl. Εννοιολογικά, είναι η ρίζα του δέντρου εγγράφου και παρέχει την κύρια πρόσβαση στα δεδομένα του εγγράφου. |
| [Element](./element/) | Η διεπαφή Element αντιπροσωπεύει ένα στοιχείο σε ένα ενσωματωμένο έγγραφο xbrl. |
| [ElementList](./elementlist/) | Η λίστα του στοιχείου. |
| [NamedNodeMap](./namednodemap/) | Αντιπροσωπεύει συλλογές χαρακτηριστικών στις οποίες μπορείτε να προσπελάσετε ονομαστικά. |
| [Node](./node/) | Η κλάση Node είναι ο κύριος τύπος δεδομένων για ολόκληρο το Μοντέλο αντικειμένου εγγράφου. Αντιπροσωπεύει έναν μόνο κόμβο στο δέντρο εγγράφων. |
| [NodeList](./nodelist/) | Η λίστα του κόμβου. |
| [ProcessingInstruction](./processinginstruction/) | Η κλάση αντιπροσωπεύει μια "εντολή επεξεργασίας", που χρησιμοποιείται στην XML ως ένας τρόπος για να διατηρούνται πληροφορίες για τον επεξεργαστή στο κείμενο του εγγράφου. |
| [Text](./text/) | Η τάξη αντιπροσωπεύει το περιεχόμενο κειμένου. |
