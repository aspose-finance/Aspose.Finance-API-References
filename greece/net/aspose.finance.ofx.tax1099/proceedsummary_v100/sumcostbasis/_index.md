## ProceedSummary_V100.SumCostBasis property

Λαμβάνει ή ορίζει τη βάση κόστους. Αυτή είναι η βάση του συνολικού κόστους για όλες τις συναλλαγές που αναφέρονται σε αυτήν τη σύνοψη. Για τις συναλλαγές με Κωδικό Α και Δ εντύπου 8949, το this θα πρέπει να είναι η βάση κόστους όπως αναφέρεται στην IRS. Για όλες τις άλλες συναλλαγές, η βάση κόστους θα πρέπει να είναι η βάση κόστους όπως είναι γνωστό από τον πάροχο.

```csharp
public string SumCostBasis { get; set; }
```
