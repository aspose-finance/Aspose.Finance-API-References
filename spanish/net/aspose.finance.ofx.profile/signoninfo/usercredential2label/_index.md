## SignonInfo.UserCredential2Label property

Obtiene o establece la solicitud de texto para la credencial de usuario. Si está presente, se requiere una cuarta credencial (USERCRED2) además de USERID, USERPASS y USERCRED1. Si está presente, UserCredential1Label también debe estar presente.
