## ProceedSummary_V100.SumCostBasis property

Obtiene o establece la base de costos. Esta es la base de costo total para todas las transacciones informadas en este resumen. Para las transacciones del Formulario 8949 Código A y D, this debe ser la base de costo según lo informado al IRS. Para todas las demás transacciones, la base de costo debe ser la base de costo conocida por el proveedor.

public string SumCostBasis { get; set; }
