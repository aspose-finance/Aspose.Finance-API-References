## ProceedSummary_V100.SumCostBasis property

Hämtar eller ställer in kostnadsbasen. Detta är den totala kostnadsbasen för alla transaktioner som rapporteras i denna sammanfattning. För Form 8949 Code A- och D-transaktioner bör this vara kostnadsbasen som rapporterats till IRS. För alla andra transaktioner bör kostnadsbasen vara kostnadsbasen som känt av leverantören.
