## Element.HasAttributeNS method

सही होता है जब किसी दिए गए स्थानीय नाम और नामस्थान यूआरआई के साथ एक विशेषता इस तत्व पर निर्दिष्ट होती है या इसका डिफ़ॉल्ट मान होता है, अन्यथा झूठा होता है।

```csharp
public bool HasAttributeNS(string namespaceURI, string localName)
```

| पैरामीटर | प्रकार | विवरण |
| --- | --- | --- |
| namespaceURI | String | विशेषता नाम स्थान URI। |
| localName | String | विशेषता स्थानीय नाम। |

### प्रतिलाभ की मात्रा

सही है अगर नामस्थान यूआरआई और स्थानीय नाम के साथ विशेषता है; अन्यथा झूठा।
